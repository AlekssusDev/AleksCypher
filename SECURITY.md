# Security Policy / Политика безопасности

## Поддерживаемые версии / Supported Versions

На данный момент активно поддерживается и обновляется только актуальная версия ветки `main`.


| Version | Supported |
| ------- | :-------: |
| 3.0.x   |     ✅    |

## Как сообщить об уязвимости / Reporting a Vulnerability

Если вы обнаружили критическую уязвимость в криптографических примитивах AleksCypher (Ackermann-RNG, Rayo-KDF и др.) или GUI-интерфейсе:

1. **Не создавайте открытый Issue**, чтобы не подвергать риску пользователей программы.
2. Пожалуйста, отправьте личное сообщение автору напрямую через приватные уязвимости GitHub (вкладка Security -> Report a vulnerability), либо свяжитесь по контактам, указанным в профиле.
3. Автор постарается исправить баг в кратчайшие сроки.
