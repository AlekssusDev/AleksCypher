# Как помочь проекту AleksCypher

Спасибо за интерес к проекту! Вот как вы можете внести вклад:

1. Форкните репозиторий.
2. Создайте новую ветку от `main` (например, `fix/readme-typo` или `feature/optional-argon2id-tuning`).
3. Внесите изменения, придерживаясь общей стилистики кода.
4. Отформатируйте код с помощью `cargo fmt`.
5. Убедитесь, что проект собирается без ошибок: `cargo build --release`.
6. Откройте Pull Request с описанием того, что и зачем вы изменили.

Для крупных изменений или идей лучше сперва открыть Issue – обсудим архитектуру до того, как вы потратите время на код.

Принятые PR будут упомянуты в истории версий.
